Minimal Node

+ TypeScript scaffold. Try the API endpoints below or follow quick start steps.

Quick start

Run locally (no Docker):

npm install
npm run seed
npm run dev

Open /docs or the endpoints listed on the right.

What this scaffold gives you

    style="line-height:1.6">
  • Simple HTTP server with endpoints: /api
    • /health, /api/version, /api/info
  • Static demo page served from public/
    • >Seed script that creates sample JSON in data/
  • Tooling: TypeScript, ESLint, Prettier, Husky, Vit
    • est and a CI template

Example usage (fetch)

fetch('/api/health').then(r => r.json()).then(console.log)

Notes

The server listens on port 3000 by default. Set PORT env to change it. The seed script writes demo JSON files into data/.

CLI / cURL

curl -s http://localhost:3000/api/health
curl -s http://localhost:3000/api/version
curl -s http://localhost:3000/api/info

If endpoints return 404, make sure the dev server is running: npm run dev.
